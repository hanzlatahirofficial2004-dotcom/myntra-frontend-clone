# 🛍️ Myntra Frontend UI Clone

A high-fidelity, pixel-perfect frontend clone of the **Myntra E-commerce web platform**. This project showcases advanced layout design, component structures, dynamic badge systems, and clean styling to replicate a production-level retail experience.

## ✨ Features
* **Pixel-Perfect Product Cards:** Beautifully engineered product components featuring brand names, descriptions, dynamic pricing structures, and dynamic discount percentage badges.
* **Smart UI Components:** Includes hover effects, custom "Add to Bag" action buttons, star-rating displays with user review counts, and a standard global navigation header.
* **Modern CSS Architectures:** Heavily utilizes Flexbox and CSS Grid alignments to ensure cards grid wraps gracefully across different viewports.
* **Clean & Semantic Codebase:** Written with scalability in mind, using structured CSS classes and reusable logic layouts.

---

## 🛠️ Tech Stack & Architecture
* **Markup:** HTML5 (Semantic headers, sections, grids, and descriptive image components)
* **Styling:** CSS3 (Advanced text-decorations for pricing models, flex distributions, hover transitions, and badge absolute positioning)
* **Interactivity:** JavaScript (Handles dynamic data structure parsing for the catalog and interface states)

---

## 📂 Core Structure Highlights
The catalog design mimics modern e-commerce engineering paradigms:
1. **Header System:** A functional global top bar housing brand categorization (`Men`, `Women`, `Kids`), a centralized global search query field, and user profile/wishlist utilities.
2. **Product Grid:** Responsive flex/grid system that dynamically scales item listings uniformly.
3. **Price Engine Visualization:** Renders standard retail price, crossed-out original price (MRP), and calculated dynamic offset percentages elegantly.

---

## 🚀 How to Run this Project Locally
1. Clone this repository to your local setup.
2. Ensure your folder contains all code assets (`.html`, `.css`, `.js`) along with your product images catalog folder.
3. Launch the main `.html` file directly using any modern standard browser (or via VS Code Live Server).
